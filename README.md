# 🎓 Student Management System

A console-based Java application developed using Eclipse IDE that performs CRUD operations (Create, Read, Update, Delete) on student records using MySQL database with JDBC connectivity.

---

## 🚀 Features

- Add student records
- View student data
- Update student details
- Delete student records

---

## 🛠️ Technologies Used

- Java
- MySQL
- JDBC
- Eclipse IDE

---

## ⚙️ How to Run

1. Open project in Eclipse
2. Connect MySQL database
3. Run main class

---

- Add authentication system (Login/Signup)
